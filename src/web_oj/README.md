# Quest Online Judge

> 基于 [Universal Online Judge 社区版](https://github.com/UniversalOJ/UOJ-System) 修改

## 特性

- 全新管理后台，操作更加丰富快捷
- 使用 AtCoder Rating System 和个性化 Rating 等级
- 丰富试题权限管理
- 增加 reCAPTCHA 验证码、注册邮箱验证、手机短信验证，提升安全性
